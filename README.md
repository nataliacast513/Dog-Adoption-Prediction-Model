# Dog Adoption Prediction Model

## Overview
This project analyzes adoption data from the Western Riverside Animal Shelter to predict whether a dog will be adopted based on its characteristics.

The analysis was conducted in **Python using Jupyter Notebook**, with a focus on exploring key drivers of adoption outcomes and comparing multiple machine learning approaches.

This project follows an end-to-end analytical workflow, including **data cleaning, exploratory analysis, model development, and model selection**, to identify patterns and build predictive insights.

This project demonstrates how predictive modeling can be used to support operational decision-making by identifying factors associated with higher adoption likelihood.

---

## Objective
The goal of this project was to:
- Predict whether a dog will be adopted based on available features  
- Identify key characteristics that influence adoption likelihood  
- Compare the performance of multiple classification models  
- Translate model findings into actionable insights  

---

## Project Workflow

### 1. Data Preparation
- Cleaned and structured raw shelter data  
- Handled missing values and inconsistent entries  
- Engineered features to improve model performance  

### 2. Exploratory Data Analysis (EDA)
- Analyzed distributions and relationships between variables  
- Identified trends in adoption outcomes  
- Highlighted key factors potentially influencing adoption rates  

### 3. Model Development
- Built and compared multiple classification models:
  - Logistic Regression (baseline, interpretable model)  
  - Decision Tree (nonlinear relationships and feature splits)  
  - K-Nearest Neighbors (distance-based classification)  

### 4. Model Evaluation & Selection
- Evaluated models using performance metrics (e.g., accuracy)  
- Compared strengths and trade-offs between approaches  
- Selected the best-performing model based on predictive performance and interpretability  

---

## Dataset
- Western Riverside Animal Shelter adoption dataset  
- Features include (examples):
  - Dog characteristics (e.g., breed, age, size)  
  - Intake and outcome attributes  
  - Other relevant adoption indicators  

Note: Data used is publicly available and analyzed for educational purposes.
